# 2.3 Understanding Plans

Arenalinq offers two plan tiers, both billed annually:

| | Scout | Program |
|---|---|---|
| **Price** | $100/year | $200/seat/year |
| **Seats** | 1 (single user) | Up to 5 seats |
| **Best for** | Individual coaches managing their own recruiting list | Coaching staffs who need shared access |
| **Pipeline stages** | Up to 4 custom stages | Unlimited stages |
| **Player limit** | Up to 1,000 players | Up to 10,000 players |
| **Player imports (CSV/Excel)** | — | ✓ |
| **Shared notes** | — | ✓ |
| **Activity history** | — | ✓ |
| **Team collaboration** | — | ✓ |

Program plan pricing is per seat — so a staff of three would be $600/year ($200 × 3 seats). You can add or remove seats at any time, and billing adjusts automatically with proration.
