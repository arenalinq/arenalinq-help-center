# 3.1 Preparing Your File

:::info Program Plan Only
Player import is available on the **Program plan** only.
:::

Arenalinq lets you import your existing recruiting lists from **CSV**, **Excel (.xlsx)**, or **legacy Excel (.xls)** files. Before uploading, make sure your file is clean and well-structured.

## Supported Fields

| Field | Notes |
|---|---|
| Email | **Required** — used for deduplication |
| First Name | |
| Last Name | |
| Full Name | Use this if your file has a single name column instead of first/last |
| Graduation Year | |
| Primary Position | |
| Phone | |
| Hometown | |
| Gender | |
| GPA | |

## Tips for a Clean Import

- Every row must have an **email address** — rows without emails will be skipped.
- Remove any blank rows or summary rows at the bottom of your spreadsheet.
- Make sure each row represents one player.
- Avoid merged cells.
- Keep your file under **10 MB** and **5,000 rows** per import.
